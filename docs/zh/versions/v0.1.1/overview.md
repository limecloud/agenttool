# Agent Tool v0.1.1

Agent Tool v0.1.1 是一个 patch release，用于在 Agent Context 成为当前已发布标准后刷新 Agent 标准生态。

## 重点

- 将 Agent Context 加入当前标准表与边界图。
- 从未来候选列表移除 Agent Context。
- 将 Agent Context 加入 README related-standard links。
- 更新 LLM entrypoints，方便 AI clients 发现 Agent Context。
- 新增 v0.1.1 版本快照页面。
