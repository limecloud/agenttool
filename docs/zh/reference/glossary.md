# 术语表

| 术语 | 含义 |
| --- | --- |
| Tool declaration | callable capability 的可移植描述。 |
| Tool surface | 某个上下文中 agent 或 model 可见的工具集合。 |
| Tool invocation | 一次被请求或已执行的工具调用。 |
| Execution profile | 工具如何运行、支持哪些 lifecycle operations。 |
| Permission profile | policy system 或 human reviewer 所需的风险与访问事实。 |
| Native id | 来自底层协议或 provider 的原生标识。 |
| Resource ref | 指向可被另一个系统读取的上下文或字节。 |
| Artifact ref | 指向由 Agent Artifact 管理的 durable deliverable。 |
| Evidence ref | 指向 trust、provenance、replay、review 或 audit facts。 |
| Policy ref | 指向 decision、approval、grant、waiver 或 risk record。 |
