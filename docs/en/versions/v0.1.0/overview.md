# v0.1.0 Overview

Agent Tool v0.1.0 is the first public draft. It defines the portable tool layer for agent systems.

## Included

- Tool declarations and tool surfaces.
- Input/output contracts and schema guidance.
- Execution profiles and permission profiles.
- Invocation lifecycle, progress, cancellation, and results.
- Error classes, retry guidance, and event classes.
- External mappings for MCP, OpenAPI, function calling, A2A, CLI, browser, and native tools.
- English and Chinese documentation.
- Public JSON Schemas.
- LLM-friendly `llms.txt` and `llms-full.txt`.
