# v0.2.0 Changelog

- Adds richer tool interface, execution pipeline, hooks, permission decision, scheduling, deferred loading, input mutation, and result persistence contracts.
- Adds examples for shell sandboxing, deferred MCP discovery, permission denial, large result persistence, and parallel read scheduling.
- Expands public JSON Schemas and LLM entrypoints.
- Updates source analysis and research sources to reflect implementation-level tool runtime analysis.
