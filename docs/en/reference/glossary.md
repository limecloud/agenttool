# Glossary

| Term | Meaning |
| --- | --- |
| Tool declaration | Portable description of a callable capability. |
| Tool surface | Context-specific set of tools visible to an agent or model. |
| Tool invocation | One requested or executed tool call. |
| Execution profile | Description of how a tool is run and which lifecycle operations it supports. |
| Permission profile | Risk and access facts needed by policy systems or human reviewers. |
| Native id | Identifier from the backing protocol or provider, preserved by Agent Tool. |
| Resource ref | Reference to context or bytes that can be fetched by another system. |
| Artifact ref | Reference to a durable deliverable governed by Agent Artifact. |
| Evidence ref | Reference to trust, provenance, replay, review, or audit facts. |
| Policy ref | Reference to a decision, approval, grant, waiver, or risk record. |
