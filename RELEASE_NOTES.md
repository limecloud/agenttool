# Agent Tool v0.1.0

Agent Tool v0.1.0 is the first public draft of a portable standard for agent tool surfaces.

## Highlights

- Defines tool declarations, tool surfaces, input/output contracts, execution profiles, and permission profiles.
- Defines invocation lifecycle, progress, cancellation, timeout, retry, result envelope, and error taxonomy.
- Preserves native ids for MCP, OpenAPI, function calling, A2A, CLI, browser, native app, and model-task tools.
- Links tool calls to Agent Runtime, Agent UI, Agent Policy, Agent Evidence, Agent Artifact, Agent Knowledge, telemetry, and peer-agent systems.
- Adds public JSON Schemas and English/Chinese documentation.
- Adds `llms.txt`, `llms-full.txt`, `llm.txt`, and `llm-full.txt` for AI clients.
